# ถอดระบบ: ระบบลาออนไลน์ผ่าน LINE

## ภาพรวม
ระบบนี้คือ Web App + LINE LIFF + LINE Messaging API สำหรับจัดการการลาในองค์กรแบบครบวงจร

## บทบาทผู้ใช้
1. พนักงาน: เชื่อมบัญชี LINE, ยื่นใบลา, ดูประวัติ, ยกเลิกใบลาที่ยังรออนุมัติ, ดูวันลาคงเหลือ
2. หัวหน้าแผนก: เห็นเฉพาะแผนกตนเอง, รับ Flex Message, อนุมัติหรือไม่อนุมัติผ่าน LINE หรือหน้าเว็บ
3. HR/Admin: จัดการพนักงาน, แผนก, ประเภทลา, วันหยุด, ผู้ใช้, รายงาน และอนุมัติขั้นสุดท้าย
4. ผู้ดูแลระบบ: ตั้งค่า LINE OA, LIFF, Webhook, Rich Menu, Trigger และนโยบายลา

## ฟังก์ชันหลัก
- ยื่นใบลาผ่านเว็บ/มือถือ/LINE LIFF
- รองรับลาป่วย ลากิจ ลาพักร้อน ลาเหตุประจำปี
- อนุมัติผ่าน LINE Flex Message
- แจ้งเตือนพนักงานและหัวหน้าอัตโนมัติ
- ดูประวัติและปฏิทินการลา
- รายงานรายบุคคล/รายแผนก/รายเดือน
- จัดการสิทธิ์และโควตาวันลา
- นำเข้าพนักงานจาก CSV
- อนุมัติหลายรายการพร้อมกันแบบ Bulk ในเฟสถัดไป

## สถานะใบลา
- PENDING: ยื่นแล้ว รอหัวหน้า
- APPROVED_L1: ผ่านหัวหน้า รอ HR
- APPROVED: อนุมัติสุดท้ายแล้ว
- REJECTED: ไม่อนุมัติ
- CANCELLED: พนักงานยกเลิก

## Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Google Apps Script
- Database: Google Sheets
- Notification: LINE Messaging API
- Mobile: LIFF
- Deploy: GitHub Pages
