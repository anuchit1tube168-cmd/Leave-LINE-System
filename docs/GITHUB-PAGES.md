# คู่มือ Deploy ด้วย GitHub Pages

1. ไปที่ repository นี้
2. เปิด Settings > Pages
3. เลือก GitHub Actions หรือ Deploy from branch
4. ถ้าเลือก Deploy from branch ให้ใช้ branch main และ folder root
5. รอระบบสร้าง URL ของ GitHub Pages
6. นำ URL ที่ได้ไปใส่เป็น Endpoint ของ LIFF

## ไฟล์สำคัญ
- index.html
- css/styles.css
- js/app.js
- apps-script/Code.gs
- data/*.csv
- docs/*.md
