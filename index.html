    ใส่ค่าจริงหลัง Deploy Apps Script / LINE
    Token และ Secret ต้องเก็บใน Apps Script Properties เท่านั้น ห้ามใส่ใน GitHub เด็ดขาด ไม่งั้นระบบกลายเป็นตู้กด Token ฟรีทันที

    คู่มือใช้งานแบบเร็ว

    1. สร้าง LINE OA + Messaging API Channel
    2. สร้าง LIFF App และใส่ Endpoint เป็น GitHub Pages URL
    3. สร้าง Google Sheet Database แล้ววาง Code.gs ใน Apps Script
    4. Deploy Apps Script เป็น Web App แล้วเอา URL ไปใส่ใน Settings
    5. ตั้งค่า Webhook URL ใน LINE Developers Console

    หัวหน้าได้รับ Flex Message เมื่อมีใบลาใหม่ กดอนุมัติหรือไม่อนุมัติจาก LINE ได้ทันที ระบบตอบกลับและบันทึกสถานะอัตโนมัติ

    Admin จัดการบัญชีผู้ใช้ แผนก หัวหน้า นำเข้า CSV ดูปฏิทิน รายงาน และอนุมัติขั้นสุดท้าย

    Supervisor เห็นเฉพาะข้อมูลในแผนกตัวเอง ระบบกรอง Dashboard, คิวอนุมัติ, ใบลา และปฏิทินให้อัตโนมัติ
